# ED-Unidad07-LIA
Actividades Unidad 07 - ED

ACTIVIDAD 01
- ParteA:
  En la Parte A de la actividad se nos indica que se debe realizar:
  -> Los actores serán el Socio que realizará las primeras acciones y el Administrador que gestionará
  -> Include/Extend: usamos include en el caso entre la identificacion y reserva (porque debe hacerse                            igualmente para poder seguir); en reserva y pagos pendientes. Por otro lado
                      usamos extend en el caso de apuntarse en las clases y el seguro ya que es opcional

  - Parte B:
    En la Parte B de la actividad se nos indica que se debe realizar:
    -> Un actor socio debe identificarse y elegir fecha/hora. Una vez itroducido los datos el sistema debe comprobar la disponibilidad. En caso de que no lo tenga se deberá elegir de nuevo OTRA fecha/hora o salir del programa. Si existe disponibilidad, el sistema debe comprobar si existen pagos pendientes. En caso de que no los haya se reserva con la fecha/hora asignada por el socio con una confirmación tipo email, de lo contrario se deniega la reserva.
